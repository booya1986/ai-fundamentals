| 🎓 **מטלת סיום + תעודה** | תכנון משימה מול הבוט "בינו" → תעודה אישית להדפסה |

## 🗂️ מבנה הקורס — 9 מודולים

| מודול | נושא | כולל |
|:---:|---|---|
| **M0** | פתיחה ואבחון | שאלון אבחון · סרטון פתיחה |
| **M1** | איך AI באמת חושב | מנוע סטטיסטי · טוקנים · attention · היררכיית AI/ML/DL/GenAI · הזיות |
| **M2** | בוחרים את הכלי הנכון | Copilot · JARVIS · NotebookLM · Napkin · RAG · רב‑מודאליות |
| **M3** | הנדסת פרומפטים · RICE | חלון ההקשר · מודל RICE · פרומפט חלש מול טוב |
| **M4** | פרודוקטיביות בעבודה | מצגות ב‑Copilot · העוזר האישי · יכולות מול מגבלות |
| **M5** | כלים מתקדמים | NotebookLM (מסמכים→ידע) · Napkin (טקסט→תרשים) |
| **M6** | אתיקה, סייבר ואבטחה | גן סגור מול עולם פתוח · הטיה · אחריות אנושית |
| **M7** | להפוך לארכיטקטים | Custom Instructions · בניית בוטים · סוכנים |
| **M8** | תרגול מסכם · בינו | מטלת סיום מקיפה → 🎓 תעודה |

> כל מודול נחתם במבחן הבנה+יישום, והתקדמות נשמרת אוטומטית (גם בהרצה עצמאית).

## 🚀 הרצה מקומית

```bash
npm install
npm run dev        # פיתוח · http://localhost:5173
npm run build      # בנייה לפרודקשן · → dist/
npm run preview    # תצוגה מקדימה של הבנייה
```

## 🧩 ארכיטקטורה

```
src/
├─ data.jsx          מבנה הקורס · מבחנים/תרגילי M0–M1 · באדג'ים · התקדמות
├─ content/m2..m7.js תוכן, מבחנים ותרגילי Copilot לכל מודול
├─ coursemap.jsx     מפת הקורס (לוח/מסע) · לוגיקת נעילה ופתיחה
├─ lesson.jsx        מסך שיעור + נגן וידאו + מריץ מבחן
├─ exercises.jsx     סוגי שאלות: רב‑ברירה · נכון/לא · התאמה · השלמה · תרחיש
├─ copilot.jsx       תרגיל "העתקה ל‑Copilot" + רפלקציה
├─ interactions.jsx  3 הסימולציות האינטראקטיביות
├─ final.jsx         מסך מטלת הסיום (בינו)
├─ gamification.jsx  תוצאות · הישגים · תעודה
├─ ui.jsx            רכיבי בסיס · אייקוני lucide אחידים · RTL
└─ scorm.js          שמירת התקדמות (SCORM אם זמין, אחרת localStorage)
```

**Built with:** React 19 · Vite · lucide-react · Noto Sans Hebrew · OKLCH design tokens.

## 📦 הגשה ל‑LMS (SCORM)

הסביבה נארזת ל‑**חבילת SCORM 2004** עצמאית (offline, עם הסרטונים והפונטים מוטמעים) לטעינה ב‑LMS של הבנק. שכבת האריזה נשמרת בנפרד מ‑repo זה, שמיועד להרצה ולמתן פידבק.
