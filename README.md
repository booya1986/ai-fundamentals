# יסודות הבינה המלאכותית — סביבת למידה (בנק הפועלים)

סביבת למידה אינטראקטיבית ומשחקית (React + Vite, עברית RTL) המאחדת את כל חומרי הקורס "יסודות הבינה המלאכותית" לחוויה אחת רציפה: סרטונים מוטמעים, סימולציות אינטראקטיביות, תרגילי "העתקה ל‑Copilot", שאלות הבנה ויישום, גיימיפיקציה (XP, רמות, באדג'ים), ומטלת סיום עם תעודה.

> זהו ה‑repo של **סביבת הקורס** לצורך הרצה ומתן פידבק. אריזת ה‑SCORM ל‑LMS נשמרת בנפרד (אינה כלולה כאן).

## הרצה מקומית
```bash
npm install
npm run dev      # פיתוח (http://localhost:5173)
npm run build    # בנייה לפרודקשן → dist/
npm run preview  # תצוגה מקדימה של הבנייה
```

## מבנה הקורס (9 מודולים)
- **M0** פתיחה ואבחון
- **M1** איך AI באמת חושב (מנוע סטטיסטי, טוקנים, attention, היררכיית AI/ML/DL/GenAI, הזיות)
- **M2** בוחרים את הכלי הנכון (Copilot · JARVIS · NotebookLM · Napkin · RAG)
- **M3** הנדסת פרומפטים · RICE
- **M4** פרודוקטיביות בעבודה (+ יכולות מול מגבלות)
- **M5** כלים מתקדמים — NotebookLM & Napkin
- **M6** אתיקה, סייבר ואבטחת מידע (+ הטיה, אחריות אנושית)
- **M7** להפוך לארכיטקטים — בניית בוטים
- **M8** תרגול מסכם עם "בינו" → תעודה

## ארכיטקטורה (src/)
- `data.jsx` — מבנה הקורס, מבחנים ותרגילי M0–M1, באדג'ים, התקדמות.
- `content/m2..m7.js` — תוכן, מבחנים ותרגילי Copilot לכל מודול.
- `coursemap.jsx` — מפת הקורס (תצוגת לוח / מסע), נעילה/פתיחה.
- `lesson.jsx` · `exercises.jsx` · `copilot.jsx` · `interactions.jsx` · `final.jsx` — מסכי שיעור/מבחן/תרגיל/סימולציות/מטלת סיום.
- `gamification.jsx` — תוצאות, הישגים, תעודה.
- `ui.jsx` — רכיבי בסיס (אייקוני lucide אחידים, RTL).
- `scorm.js` — שמירת התקדמות (SCORM אם זמין, אחרת localStorage).

ההתקדמות נשמרת מקומית (localStorage) בהרצה עצמאית, כך שאפשר להתנסות בכל המסלול ולתת פידבק.
